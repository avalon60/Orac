- Include the normal script header used in this project.
- Use argparse for CLI arguments.

- Return exit code 0 when clean and 1 when issues are found.

- docs/agent-guardrails exists.

- plain references such as docs/agent-guardrails/foo.md

- https:// links
- anchors-only links

- external Markdown links
